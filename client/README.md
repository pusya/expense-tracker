# Expense Tracker

React MERN Stack CRUD application. A online app where you can register, login, and record your expense.

## Getting Started

```bash
npm install
npm client-install
npm run dev
```

## Components Structure

```bash
App
|___Home___________
|___About          |____ItemForm
|___Register       |____ItemFilter
|___Login          |____Total
                      |
                      |____Items
                              |
                              |____SingleItem
                                        |
                                        |____Edit Btn
                                        |____Delete Btn
```
